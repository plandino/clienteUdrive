Documentación del ambiente de desarrollo¶

El producto fue desarrollado en Ubuntu 14.04 de 64 bits. Se uso el IDE Android Studio 1.4, con Java 8, con la version de Gradle 2.4. La aplicación fue desarrollada para Android 4.4.2 (SDK 19) como minimo y para Android 5.1.1 (SDK 22) como maximo. No se desarrollo para Android 6.0 (SDK 23) debido a que esto incluia cambiar varios detalles de permisos y clases que usa Android 6.0, pero que general conflicto con versiones anteriores.

Ver Documentacion de las librerias y la compilacion por demás requisitos del sistema, puesto que debe permitir la instalación de todas las dependencias.
